## 1. Главная страница: Карта и быстрый поиск

Центральный элемент всего сервиса. Пользователь попадает сюда и сразу видит карту с объектами вокруг себя.

### Блок 1.1: Шапка с умным поиском
- **Строка поиска:** Поле с автодополнением. Пользователь вводит: *«воркаут», «бассейн с сауной», «купить гантели», «Asics Kayano 31»*.
- **Фильтр «Мне нужно сейчас»:** Кнопки быстрого выбора:
    - 🏋️ *Позаниматься* (залы и площадки)
    - 🏊 *Поплавать* (бассейны)
    - 🛒 *Купить* (магазины)
- **Кнопка геолокации:** Центрирует карту на пользователе.

### Блок 1.2: Интерактивная карта (Основной холст)
- **Пины на карте:** Цветовое кодирование по типу объекта:
    - 🔴 Красный — Тренажерный зал
    - 🔵 Синий — Бассейн
    - 🟢 Зеленый — Спортплощадка (воркаут/футбол/баскетбол)
    - 🟡 Желтый — Спортмагазин
- **Динамическая иконка загруженности (только для залов):** Если данных достаточно, пин может иметь индикатор:
    - 🟢 Зеленый ободок — свободно
    - 🟡 Желтый — средне
    - 🔴 Красный — забито
- **Клик по пину:** Открывает превью-карточку объекта (краткая инфа + кнопка «Подробнее»).

### Блок 1.3: Панель фильтров (слева или снизу в мобильной версии)
Раскрывающийся набор фильтров, который сужает видимые на карте пины.

**Фильтры по категориям:**
- **Тип:** `Все` / `Залы` / `Площадки` / `Бассейны` / `Магазины`
- **Цена:** `Бесплатно` / `До 500₽ разовое` / `Абонемент до 2000₽/мес` / `Премиум`
- **Режим работы:** `Открыто сейчас` / `Круглосуточно` / `Утро (до 12)` / `Вечер (после 18)`
- **Удобства:** `Душ` / `Сауна` / `Парковка` / `Доступность для инвалидов`
- **Специфика (для площадок):** `Освещение` / `Резиновое покрытие` / `Сетки на воротах`

**Умный фильтр «Сейчас свободно»:**
Отдельная галочка, которая показывает только те залы, где по данным пользователей сейчас низкая загруженность.

## 2. Страница детальной карточки объекта

Открывается при клике на «Подробнее» из превью на карте. Содержит всю информацию, структурированную по вкладкам.

### Вкладка 2.1: Общая информация (для всех типов)
- **Карусель фото:** Официальные фото от владельца + фото от пользователей (с сортировкой по свежести).
- **Блок контактов:** Адрес, телефон, сайт, соцсети. Кнопка «Построить маршрут».
- **Блок «Часы работы»:** Таблица с днями недели. Плашка `🟢 Открыто сейчас` или `🔴 Закрыто`.
- **Блок «Цены»:** Актуальная стоимость разового посещения, абонементов (с указанием источника данных: «Подтверждено администрацией» или «По данным пользователя @Ivan»).
- **Блок «Оценка и отзывы»:** Средний рейтинг по 5-балльной шкале. Отзывы с фильтром по тегам («чисто», «старые тренажеры», «тесная раздевалка»).

### Вкладка 2.2: Специфическая информация (зависит от типа объекта)

**Для Тренажерного зала:**
- **График загруженности:** Виджет на основе истории посещений. Столбчатая диаграмма по дням недели и часам с цветовой шкалой (зеленый-красный).
- **Наличие тренера:** Индикатор `🟢 Тренер в зале` (отмечают посетители в реальном времени).
- **Зоны:** Галочки `Кардио`, `Свободные веса`, `Тренажеры`, `Кроссфит / Функционал`, `Зона растяжки`.
- **Пробное занятие:** Текстовая информация («Бесплатно при первом посещении, нужен паспорт»).

**Для Спортплощадки (воркаут/футбол/баскетбол):**
- **Тип площадки:** `Воркаут` / `Футбол` / `Баскетбол` / `Волейбол` / `Универсальная`.
- **Состояние:** Оценка покрытия (1-5), оценка чистоты.
- **Оснащение:**
    - 💡 Освещение: `Есть / Нет / Есть, но не работает`
    - 🥅 Сетки/Кольца: `В наличии / Отсутствуют / Порваны`
    - 🏋️ Турники/Брусья: `Исправны / Есть сломанные (флаг)`
- **Доступ:** `Свободный` / `По записи` / `Платная аренда`.

**Для Бассейна:**
- **Расписание свободного плавания:** Таблица на неделю с указанием дорожек. Кнопка **«Добавить в календарь»** (экспорт в .ics).
- **Параметры чаши:** Длина дорожек (25/50м), глубина, температура воды (по данным пользователей с термометром).
- **Инфраструктура:** Вышка, детская зона, гидромассаж, сауна/хамам.
- **Медсправка:** Требуется / Не требуется. Ссылка на ближайшие медцентры, где делают справку в бассейн.

**Для Спортмагазина:**
- **Поиск по товарам:** Поле ввода «Что ищете?». Выпадающий список с товарами, которые, по данным пользователей, есть в наличии в этом конкретном магазине.
- **Бренды:** Список представленных брендов.
- **Услуги:** `Примерка`, `Доставка`, `Самовывоз`.
- **Актуальные цены:** Лента фотографий ценников от пользователей с датой загрузки.

### Вкладка 2.3: Краудсорсинговые обновления (Лента статуса)
Хронология пользовательских отчетов об изменениях на объекте:
- *«15.05.2026, 18:30, @Антон: Свет на площадке не работает»*
- *«12.05.2026, 10:15, @Мария: Цена на безлимит выросла до 2500₽»*
- *«05.05.2026, 09:00, @Admin: Информация подтверждена администрацией зала»*

### Блок 2.4: Кнопки действий
- **«Я здесь / Отметить загруженность»** — Пользователь на месте может ткнуть: `🟢 Свободно`, `🟡 Средне`, `🔴 Забито`.
- **«Сообщить об изменении»** — Кнопка, открывающая форму быстрого репорта (см. раздел 5).
- **«В избранное»** — Сохранить объект для быстрого доступа.

## 3. Страница «Добавить новый объект»

Страница для краудсорсингового наполнения базы.

### Блок 3.1: Указание точки на карте
- Интерактивная карта, где пользователь ставит маркер в нужном месте.
- Кнопка «Определить адрес по метке» (обратное геокодирование).

### Блок 3.2: Форма добавления
- **Тип объекта:** Выпадающий список (Зал / Площадка / Бассейн / Магазин).
- **Название:** Текстовое поле.
- **Фотографии:** Загрузка до 5 фото (с возможностью сделать фото сразу на камеру телефона).
- **Поля в зависимости от типа:** Динамически подгружаются специфические поля из раздела 2.2.
- **Кнопка «Отправить на модерацию»**.

## 4. Личный кабинет пользователя

### Блок 4.1: Профиль и статистика
- Имя, аватар.
- **Уровень доверия:** `Новичок` → `Проверенный` → `Эксперт` (зависит от количества подтвержденных репортов).
- **Бейджи:** «5 подтвержденных обновлений», «Добавил 3 новых объекта».

### Блок 4.2: Мои объекты и репорты
- Список добавленных пользователем мест со статусом модерации.
- История отправленных репортов об изменениях с результатом: «Подтверждено» или «Отклонено».

### Блок 4.3: Избранное
- Сетка сохраненных залов, бассейнов и площадок для быстрого перехода к их карточкам.

## 5. Страница (модальное окно) «Быстрый репорт об изменении»

Вызывается из карточки объекта. Цель — минимизировать время на отправку полезной информации.

### Элементы формы:
1.  **Тип репорта (выбор иконкой):**
    - 🏋️ *Загруженность* (выбор Свободно/Средне/Забито)
    - 💡 *Освещение* (Работает/Не работает)
    - 🛠️ *Поломка* (Указать элемент: турник, дверь, душ)
    - 💰 *Цена изменилась* (Поле для ввода новой цены)
    - ⏰ *Расписание изменилось* (Текстовое поле)
2.  **Фото (опционально):** Сделать снимок для подтверждения.
3.  **Комментарий:** Текстовое поле.
4.  **Кнопка «Отправить»**.

## 6. Страница модерации (для администраторов)

Служебный раздел для обработки пользовательского контента.

### Блок 6.1: Очередь на модерацию
- **Вкладки:** «Новые объекты», «Репорты об изменениях», «Новые фото».
- **Карточка репорта:**
    - Слева: Текущая информация в базе.
    - Справа: Предлагаемое пользователем изменение (с фото).
    - Кнопки: **«Принять»**, **«Отклонить»**, **«Запросить уточнение»**.

### Блок 6.2: Управление пользователями
- Список пользователей с рейтингом доверия.
- Возможность повысить/понизить уровень доверия вручную.
- Бан за спам или ложные репорты.

## 7. Технические аспекты реализации

### 7.1. Архитектура базы данных (упрощенно)

**Таблица `objects`:**
- `id`, `type` (enum), `name`, `lat`, `lng`, `address`, `phone`, `website`, `working_hours` (JSON), `created_by`, `status` (active/pending/archived).

**Таблица `object_details`:**
- Хранит специфические поля в формате JSON или отдельными колонками с разреженной структурой.

**Таблица `reports`:**
- `id`, `object_id`, `user_id`, `report_type`, `old_value`, `new_value`, `photo_url`, `status` (pending/approved/rejected), `created_at`.

**Таблица `crowd_metrics`:**
- `id`, `object_id`, `user_id`, `metric_type` (occupancy/broken_equipment), `value`, `created_at`. Используется для построения графиков загруженности.

### 7.2. Алгоритм расчета загруженности
- Учитываются только репорты с типом «Загруженность» за последние 2 часа.
- Если репортов меньше 3 — данные считаются недостоверными, индикатор не показывается.
- Средневзвешенная оценка с учетом времени (чем свежее репорт, тем выше вес).

### 7.3. Система доверия пользователей
- **Уровень 0 (Новичок):** Все репорты идут на ручную модерацию.
- **Уровень 1 (Проверенный):** 5+ подтвержденных репортов. Изменения применяются сразу, но попадают в лог для пост-модерации.
- **Уровень 2 (Эксперт):** 50+ подтвержденных репортов. Возможность править основные поля объекта без премодерации.

### 7.4. Интеграции
- **Геокодирование:** API Яндекс.Карт, Google Maps или OpenStreetMap (Nominatim).
- **Проксирование фото:** Все пользовательские фото пропускаются через сервер для сжатия и удаления EXIF-метаданных (приватность).
- **Push-уведомления:** Для оповещения о подтверждении репорта или изменении статуса любимого зала.

## 8. UX-фишки для вовлечения

- **Кнопка «Заняться спортом через 10 минут»:** На главном экране большая зеленая кнопка. По нажатию — показывает 3 ближайшие **открытые и свободные** локации для занятий (зал или площадка) в радиусе 15 минут пешком.
- **«Спортивный маршрут»:** Пользователь выбирает «Хочу на турники», приложение строит пеший маршрут до ближайшей бесплатной воркаут-площадки с хорошим рейтингом и рабочим освещением.
- **Виджет сравнения бассейнов:** На странице списка бассейнов — компактная таблица сравнения по цене за час свободного плавания и времени в пути от дома/работы пользователя.
- **Уведомление «По пути»:** Если пользователь сохранил спортмагазин в избранное и проходит мимо него (геофенсинг), приходит пуш: *«Вы рядом с Кантом. Проверить наличие Asics Kayano 31?»*
